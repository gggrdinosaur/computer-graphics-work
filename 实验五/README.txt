5、实验总结
通过本实验，掌握了圆的中点Bresenham算法，能够根据半径基于八分法画出完整的圆。

本次实验遇到了较多问题，前面实验画多边形时采用的数组形式，但在本次实验中行不通，需要上网搜索如何根据点画出多边形。
搜索得知，采用glBegin(GL_LINE_LOOP)：可以绘制从第一个顶点到最后一个顶点依次相连的一组线段，然后最后一个顶点和第一个顶点相连，
第n和n+1个顶点定义了线段n，绘制n条线段;而glBgin(GL_POLYGON) 可以 绘制了一个凸多边形，顶点1到n定义了这个多边形。
四联通和八联通都采用递归的形式，在程序运行时能够感受到算法的快慢，相对于四联通来说，八联通算法效率更低，且通过比较，
泛滥填充算法时间最少，也是最快的。
